# Project Instructions for Codex

Project type:
Simple static website.

Stack:
- HTML
- Tailwind CSS
- Content from .txt files

Goals:
Update sections of the page using information from text files and keep the design consistent.

Rules:

1. Be concise and practical.
2. When editing a section, return only the HTML block ready to paste.
3. Do not include long explanations unless explicitly requested.
4. Keep the design sober, modern and elegant.
5. Use Tailwind utility classes already present in the project when possible.
6. Do not introduce JavaScript frameworks.
7. Do not change the structure of unrelated parts of the page.
8. When reading .txt files, summarize clearly and avoid inventing information.
9. Prefer semantic HTML.
10. Keep layouts simple and responsive.
11. Responde siempre en español neutro

Style guidelines:

- clean spacing
- good typography
- balanced margins
- responsive layout
- readable sections

Performance:

- Avoid scanning the whole project unless necessary.
- Only read files relevant to the task.
